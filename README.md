# 🐕 Espoir Canin - Site Web Officiel

![Espoir Canin Banner](assets/images/hero-home-ai.jpg)

## 📝 Description

Site vitrine professionnel pour **Espoir Canin**, service d'éducation et de rééducation canine basé à Natzwiller (67130) en Alsace, France.

Ce site présente les différentes prestations proposées par David Koessler, éducateur canin professionnel utilisant des méthodes positives et bienveillantes.

### 🌐 URL de Production
**[https://espoir-canin.fr/](https://espoir-canin.fr/)**

---

## ✨ Fonctionnalités

- 📱 **Design Responsive** - Compatible mobile, tablette et desktop
- 🎨 **Thème sombre moderne** - Interface élégante avec animations subtiles
- 📅 **Planning dynamique** - Chargement JSON des événements à venir
- 📧 **Formulaire de contact** - Intégration Formspree
- 🗺️ **Carte interactive** - Localisation Google Maps intégrée
- ⚡ **Performances optimisées** - Images optimisées, CSS minimal

---

## 🛠️ Stack Technique

| Technologie | Usage |
|-------------|-------|
| **HTML5** | Structure sémantique |
| **CSS3** | Styles, animations, variables CSS |
| **JavaScript** | Interactions, menu mobile, chargement planning |
| **Google Fonts** | Typographies (Inter, Outfit) |
| **Formspree** | Gestion du formulaire de contact |
| **Google Maps** | Carte de localisation |

```

│   ├── js/main.js       # Scripts JavaScript
│   ├── images/          # Images optimisées
│   ├── docs/            # Documents téléchargeables (CGV)
│   ├── events.json      # Données du planning
│   └── php/             # PHPMailer (backup)
└── data/
    └── events.json      # Événements du planning
```

## 🚀 Installation & Développement

### Prérequis
- Un serveur web local (Live Server, XAMPP, etc.)
- Navigateur moderne (Chrome, Firefox, Edge, Safari)

### Lancer le site en local
1. Cloner le repository
2. Ouvrir `index.html` avec un serveur local
3. Le site est accessible sur `localhost:5500` (Live Server) ou équivalent

```bash
# Avec Live Server (VS Code Extension)
# Clic droit sur index.html → "Open with Live Server"
```
